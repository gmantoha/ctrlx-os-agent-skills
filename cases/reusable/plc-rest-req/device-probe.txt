=== REST API (Storage) ===

Endpunkt:  GET  /storage/api/v1/media
           POST /storage/api/v1/tasks

Payload POST (entspricht altem REST-Ansatz, jetzt via DL bevorzugt):
  {"action":"mount","parameters":{"media":"3C20-2F7A","device":"sdb1","assignment":"data-exchange"}}

ACHTUNG: Im REST-API-Payload heißt das Feld "assignment" mit Kleinbuchstaben ("data-exchange"),
         im Data-Layer-Schema wird der Enum-Wert "DataExchange" (CamelCase) verwendet.
         Beide Wege führen zum selben Ergebnis.
